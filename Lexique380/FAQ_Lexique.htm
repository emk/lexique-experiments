Où se trouve la base Lemmes.txt que l'on avait dans Lexique 2?

Il faut aller dans C:\Lexique\Bases+Scripts. Vous y trouverez un fichier lemmes.bat. Double-cliquez dessus. Attendez que la fenêtre disparaisse. Vous avez maintenant un fichier Lex3.lemmes.txt dans le dossier Bases+Scripts

Lexique permet-il d'avoir la fréquence de deux homonymes ou homophones (la danse-je danse; les fils-le fils) ?

Oui, à partir de Lexique 3 il suffit de trouver l'entrée correspondante.

Comment définir un mot de basse ou de haute fréquence?

Si votre recherche se fonde sur des travaux publiés, c'est un bonne idee d'utiliser les memes critères que ceux-ci.
Quel que soit le critere retenu,
- verifiez les mots: les intuitions de fréquence sont importantes aussi, et les valeurs fournies par lexique peuvent être biaisées par le corpus utilisé.
- plus la différence de fréquence sera importante, mieux ce sera.
Maximiser votre différence de fréquence en fonction des autres contraintes que vous désirez manipuler. - si vous avez beaucoup de contraintes externes, je dirais d'expérience (mais ça n'engage que moi, à vous de vous faire une idée) que <5 = très rares, <10 = rares >20 = fréquent >50 = très fréquents

Qu'est ce qu'Open Lexique ?

Open Lexique est accessible dans la rubrique Recherche dans Bases.
Il permet déjà de faire des requêtes simultanément sur Graphemes, et/ou Lemmes et/ou Surface.et/ou d'autres bases de données. Open Lexique devrait permettre de faire évoluer Lexique continuellement en ajoutant régulièrement de nouvelles bases.
Bonnes Recherches!!!!!

J'ai copié/collé mes résultats dans Excel mais tout est noir!!!

C'est du à un problème d'Excel 97 ou 95 (ça ne le fait pas sous Excel 2000). Il vous suffit pour voir vos données de faire "Edition | Selectionner Tout" et de changer le fond en blanc (cliquez sur le pot de peinture et sélectionnez "Aucun Remplissage").

J'ai copié/collé mes résultats dans Excel mais je n'obtiens pas un tableau!!!

Vous devez utiliser "Internet Explorer" pour que le copier/coller se réalise bien.

A quoi sert la mailing-liste ? Comment s'y inscrire ?

S'inscrire à la mailing-liste vous permet de recevoir un email quand une nouvelle annonce concernant les derniers développements de Lexique est postée.
Pour vous inscrire, il vous suffit de mettre votre email dans la boîte "Mailing-List du forum" à gauche et de cliquer sur "S'inscrire". Votre email restera confidentiel.

    Comment interroger Lexique autrement?

    Le mieux est d'installer gawk, perl, les textutils (maintenant appelés aussi coreutils), et bash.
    -Pour les utilisateurs de systèmes Unix, ces utilitaires doivent être déjà installés.
    -Pour les utilisateurs de Mac OS X, il vous faut installer Fink qui vous permettra d'installer ces utilitaires
    -Pour les utilisateurs de Windows, il faut tout d'abord que vous installiez Undows.
  • Ensuite, vous pouvez au choix:
    1. Cliquer directement sur "Undows" (Dans le menu Démarrer | Programmes | Lexique)
    2. Ouvrir le dossier "c:\lexique", faire un clic droit sur le dossier "Bases+Scripts", puis sélectionner "Shell To Unix".
  • Une "boite de commandes" s'ouvre alors dans laquelle vous pouvez taper des "commandes". Par exemple, tapez:
    gawk '$1~/aux$/' Graphemes.txt >Resultats.txt
    Appuyez sur la touche "Entrée" pour éxécuter la commande. Les résultats se trouvent alors dans le fichier texte "Resultats.txt"
    Vous y verrez toutes les lignes dont les mots de la première colonne de Graphemes.txt finissent par "aux".

    Pour d'autres exemples de requêtes, lisez la FAQ d'Undows.
